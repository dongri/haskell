ラムダ (Lambda)

Lambda(無名関数)

Pythonの例

def plus(x, y):
    return x + y
a = plus(2, 4)
print(a)

plus = lambda x, y: x + y
print(plus(2,4))

関数を定義しなくてよくなる

# funcの場合
lis = [2, -10, 5, -3, 6, 7, -8]
def func(n):
    return n**2 * (1, -1)[n < 0]
lis = list(map(func, lis))
print(lis)

# lambdaの場合
lis = [2, -10, 5, -3, 6, 7, -8]
lis = list(map(lambda x: x**2 * (1, -1)[x < 0], lis))
print(lis)
